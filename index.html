Cómo funciona

Cuatro pasos,
sin complicaciones

La extensión automatiza el marcado de asistencia dentro del sistema SIU Guaraní.

01

Abrí la extensión

Usala directamente sobre la página del SIU Guaraní.

02

Pegá los DNIs

Ingresá la lista de alumnos presentes.

03

Procesamiento automático

La extensión identifica y marca los alumnos automáticamente.

04

Formulario opcional

También podés generar formularios asociados a tu cuenta de Google.

Funcionalidades

Integración académica y automatización

Diseñada para simplificar tareas docentes y administrativas universitarias.

📋

Procesamiento de DNIs

Procesamiento rápido de listas completas de alumnos.

⚡

Automatización

Marcado automático dentro de SIU Guaraní.

📝

Google Forms

Generación opcional de formularios asociados al usuario autenticado.

🔒

OAuth seguro

Autenticación mediante Google OAuth 2.0 con permisos limitados.

Uso de Google APIs

Algunas funcionalidades opcionales utilizan Google OAuth 2.0 para crear formularios asociados a la cuenta del usuario.

  • Google Drive API
  • Google Forms API

La aplicación utiliza únicamente los permisos mínimos necesarios para ejecutar las funcionalidades solicitadas por el usuario.

🔒

Privacidad y protección de datos

La aplicación no vende información personal ni utiliza los datos para publicidad o seguimiento. El procesamiento ocurre principalmente de manera local en el navegador del usuario.
