Política de Privacidad

🛡️

Sin recolección masiva

No recopilamos ni comercializamos datos personales de usuarios o estudiantes.

💻

Procesamiento local

La mayor parte del procesamiento ocurre localmente dentro del navegador del usuario.

🔑

Acceso limitado

Los permisos de Google OAuth se utilizan únicamente para las funciones explícitamente autorizadas por el usuario.

👤

1. Responsable de la aplicación

La extensión Asistencia SIU + Form al Toque fue desarrollada por Hernán Coniglio, Mauricio Pintos y Matías Müller, integrantes de la comunidad académica de la Universidad Nacional de Hurlingham (UNAHUR), Argentina.

Esta herramienta tiene como objetivo facilitar tareas administrativas y académicas relacionadas con el sistema SIU Guaraní.

Para consultas relacionadas con privacidad o uso de datos: hernanconiglio@gmail.com

📄

2. Información procesada por la extensión

La extensión puede procesar la siguiente información únicamente con fines funcionales:

  • Números de DNI ingresados manualmente por el docente.
  • Información visible dentro de páginas del sistema SIU Guaraní abiertas por el usuario.
  • Datos mínimos necesarios para generar formularios asociados a la cuenta de Google del usuario, cuando dicha funcionalidad es utilizada.
⚠️ Toda la información se utiliza exclusivamente para ejecutar las funciones solicitadas por el usuario.
🔑

3. Uso de Google APIs y OAuth

Algunas funcionalidades opcionales requieren autenticación mediante Google OAuth 2.0.

En caso de que el usuario otorgue autorización, la aplicación puede utilizar:

  • Google Drive API
  • Google Forms API

El acceso se utiliza exclusivamente para:

  • Crear formularios asociados a la cuenta del usuario.
  • Permitir que el usuario gestione dichos formularios.

La aplicación no accede, lee, modifica ni comparte correos electrónicos, documentos personales, contactos u otra información no relacionada con la funcionalidad declarada.

El uso y transferencia de información recibida desde Google APIs hacia cualquier otra aplicación cumple con la Google API Services User Data Policy, incluyendo los requisitos de Limited Use.
🚫

4. Datos que NO recopilamos

  • No vendemos información personal.
  • No utilizamos datos para publicidad.
  • No realizamos perfiles de usuarios.
  • No compartimos información con terceros.
  • No utilizamos herramientas de tracking, analytics invasivos ni fingerprinting.
  • No almacenamos bases de datos de alumnos o docentes en servidores externos.
💾

5. Almacenamiento local

La extensión puede utilizar almacenamiento local del navegador (chrome.storage.local) únicamente para guardar:

  • Preferencias del usuario.
  • Configuraciones de funcionamiento.
  • Datos temporales necesarios para la operación local.

Esta información permanece exclusivamente en el dispositivo del usuario y no es transmitida a servidores externos.

🧹

6. Eliminación y revocación de permisos

El usuario puede revocar el acceso OAuth otorgado a la aplicación en cualquier momento desde:

https://myaccount.google.com/permissions

También puede eliminar los datos almacenados localmente desinstalando la extensión o limpiando los datos del navegador.

👶

7. Uso destinado

Esta herramienta está orientada a docentes, instituciones educativas y usuarios mayores de 18 años.

📝

8. Cambios en esta política

Esta política puede actualizarse periódicamente para reflejar cambios técnicos, legales o funcionales.

Las modificaciones serán publicadas en esta misma página.
